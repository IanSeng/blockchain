# blockchain

## Getting Started

### Prerequisites 
* npm 

## Development setup 
Clone this project to your machine 
Go into a terminal window in the root directory of this repository and run 
* 
```
npm install 
npm run dev 
``` 

## Start up the project 

```
npm run start 
```

## To run test environment 
Go into a terminal window in the root directory of this repository and run 

```
npm install 
npm run test 
```
